New features and functionality

The latest release of ThoughtSpot cloud contains new and enhanced features.

Scriptability for Views

You can now import and export Views. This functionality allows you to download a View as a flat TSL file, optionally modify it, and then upload it to the same or another cluster.

See Scriptability.

Overwrite existing Answers and Pinboards with Scriptability

You can now replace an existing Answer and Pinboard with a modified TSL file. This functionality allows you to update existing Answers and Pinboards using Scriptability, without creating a new object.

See Scriptability.

Specify persona during onboarding

You can now choose your persona during ThoughtSpot onboarding. This functionality allows new users to specify whether they are an analyst or business user when they first use the product. Ten, ThoughtSpot takes the user through the appropriate onboarding experience for their persona. Existing users can also revisit onboarding to specify their persona.

See Analyst onboarding and Business user onboarding.
